# Labyrinth Support API

Backend-сервис для поддержки игрового процесса 2D roguelite dungeon crawler.

Проект реализует базовую игровую логику:
- регистрация и авторизация игроков (JWT)
- хранение профиля и прогресса
- запуск и завершение вылазок (runs)
- запись игровых событий
- лидерборд
- история действий игрока

---

## 🚀 Технологии

- Backend: Flask
- База данных: MySQL 8.0
- Аутентификация: JWT
- Клиент: HTML + JS (test_system)

## 🔑 Основные эндпоинты

### Авторизация
- `POST /auth/register`
- `POST /auth/login`

### Профиль
- `GET /profile/`

### Вылазки (Run)
- `POST /run/start`
- `POST /run/finish`

### События
- `POST /events/`
- `GET /events/`

### Лидерборд
- `GET /leaderboard/`

---

## 🧪 Тестирование

Для тестирования используется HTML-клиент:

```
test_system/
```

Страницы:
- регистрация / авторизация
- меню игрока
- вылазка (симуляция игры)
- лидерборд
- история событий

---

## 🎮 Игровая логика (упрощённо)

1. Игрок авторизуется
2. Запускает вылазку (`/run/start`)
3. Отправляет события (`/events/`)
4. Завершает вылазку (`/run/finish`)
5. Получает награды (XP, валюта)
6. Результат влияет на лидерборд
